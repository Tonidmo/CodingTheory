Reed-Muller Codes

CodingTheory.ReedMullerMod.ReedMullerCode — Type
ReedMullerCode(q::Integer, r::Integer, m::Integer, verify::Bool=true)

Return the RM(r, m) Reed-Muller code over GF(q).

If the optional parameter verify is set to true, basic checks are done to ensure correctness.
